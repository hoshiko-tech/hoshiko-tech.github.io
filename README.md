# hoshiko.tech

Company website for Hoshiko Tech Ltd. — https://www.hoshiko.tech

Static site served via GitHub Pages. Styled with Tailwind CSS (loaded from CDN), no build step.

## Local preview

Open `index.html` directly in a browser, or serve the folder with any static server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000.
